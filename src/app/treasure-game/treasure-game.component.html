遊戲規則

1. 遊戲開始, 使用EOS參與遊戲

2. 每參與一次, 將會獲得一個編號 (如果您參與2次, 將獲得2個編號)

3. 如果參與人次已滿, 開啟寶箱將會隨機抽取一個編號, 對應的參與者獲得該獎池獎勵, 遊戲結束

4. 如果參與人次未滿, 將會進入倒計時, 如果在本輪倒計時結束前有新的參與者, 將會刷新倒計時重新開始計時; 如果本輪倒計時結束時沒有新的參與者, 最後購買者將會獲得該獎池獎勵, 遊戲結束

5. 重啟遊戲, 開始下一局奪寶

6. 重啟遊戲, 開啟寶箱的用戶均可獲得0.05EOS獎勵; 平台收取總投入額的10%作為平台維護費
